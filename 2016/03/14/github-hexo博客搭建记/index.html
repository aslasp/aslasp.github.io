github+hexo博客搭建记

发表于 2016-03-14   |  

利用github pages + hexo 搭建个人博客是一件非常简单的事情。下面记录一下主要流程

在github上新建一个repository

假设你已经拥有github账号，那么首先来新建一个repo吧。

注意：repository name 必须为 你的ID.github.io

安装Hexo

Hexo是一个简单、快速、强大的Node.js静态博客框架。你当然可以选择不使用它，直接编写html+css+js代码来写博客，只是那样有点太累了=。=

安装Hexo分以下几个步骤：

安装node.js和npm

这个不用讲了吧，去node.js官网下一个装上就行。

现在的node.js都自带npm。你可以在命令行输入node -v和npm -v看看是否安装成功。

（可选）为npm设置淘宝NPM镜像

因为众所周知的原因，国内访问npm可能不太稳定，所以用淘宝镜像比较好。

使用方法见链接。

安装hexo

npm install -g hexo-cli

使用hexo搭建博客

初始化工程

mkdir directory
cd directory
hexo init
npm install hexo-server --save

编辑配置文件

打开_config.yml文件，设置好相关信息（博客名、author等），关键是下面这一段

deploy:
type: git
repo: 你刚才建的repository的地址，推荐用SSH
branch: master

配置完成后，博客就算搭好了。

正常使用

正常使用时一般有以下几个步骤：写博客，生成博客，线下测试，部署到github。

写博客

hexo new "title"

这样就创建了一个新的md文件，使用markdown语法编写你的博客。

生成

推荐这样：

hexo clean
hexo g

线下测试

hexo s

然后就可以去指定端口预览你的博客了。

部署到github

hexo d

完成！
